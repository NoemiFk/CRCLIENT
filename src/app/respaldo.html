En esta etapa se debe realizar el mapeo de los campos del portafolio. El mapeo es necesario para seleccionar los campos que se requieren para las diferentes funcionalidades; recomendación para el Mapeo:


1. Análisis y Segmentación: Seleccionar campos como Cartera Vencida, Por vencer, Fecha último pago, Estado, Producto evitar campos como nombre del cliente, dirección, Fecha de Nacimiento.

2. Comunicación: Seleccionar campos para personalización como nombre, dirección, cartera vencida, numero de cliente o contrato, referencia de pago

3. Portal: Seleccionar campos que el cliente puede visualizar como el desglose de su cartera vencida, nombre, numero de cliente o contrato, referencia de pago.

4. Aval: Seleccionar campos relacionados con el aval, Nombre, Dirección.

5. El formato del campo debe ser editado en la columna “Validación” en los casos como por ejemplo el teléfono o email.

IMPORTANTE: Se debe considerar seleccionar el formato “Id Unico” para aquel campo que va a ser el identificador en todos los pasos de gestión, estos podrían ser: No. de Contrato, No. de Cliente etc.
